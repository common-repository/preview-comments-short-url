== Description ==

Allows you to expand and preview short URL's in comments, useful to detect spam not caught by your antispam plugin or links to malicious sites before comment approval.

Uses jQuery to retrieve the long URL from [longurl.org](http://longurl.org/).

== Installation ==

* Upload the file and extract it in the `/wp-content/plugins/` directory
* Activate the plugin through the 'Plugins' menu in WordPress
* There is no step 3
